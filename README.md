# Tic Tac Toe Game

## Objective
Develop a simple Tic Tac Toe game to practice JavaScript logic and DOM manipulation.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Two-player game
- Win and draw detection
- Interactive UI

## Description
This project demonstrates basic game logic implementation and user interaction using JavaScript.

## Outcome
Improved understanding of event handling, conditional logic, and DOM manipulation.
